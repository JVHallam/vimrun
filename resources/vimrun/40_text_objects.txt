            ( { or } )


  With the { and } operators.
  You can quickly jump through paragraphs of text.

  It's a super useful operator.

  Personally, i'm quite lazy. Therefore,
  it's so much easier for me to bash the 
  } key repeatedly, so that i can move through text
  instead of using the jump queue.

           ( ( and ) )


  This is a sentence. This sentence follows. Then this one.
  This is also
  a sentence.
  You can jump through them using the ( and ) keys. They're useful.
  A. Sentence. Is. Denoted. By. A. Full. Stop.

  They're super useful keys, especially when you're being super lazy.

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

           (  d, a, s  )


--> This is sentence one. XXX XXX XXX XXX. This is sentence two.
==> This is sentence one. This is sentence two.

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

           (  c, i, s  )


--> This is sentence one. XXX XXX XXX XXX XXXX. This is sentence three. 
==> This is sentence one. This is sentence two. This is sentence three. 

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

           (  v, a, s  )


--> This is a good sentence. 
==> This is a good sentence. This is a good sentence. This is a good sentence.
